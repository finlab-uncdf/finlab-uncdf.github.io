Mobile Interface Design Best Practices

The Mobile Interface Design Best Practices are a set of guidelines which can help participants design mobile interfaces while creating testable prototypes. These can also help inform the design and flow of the final product.

USE CASES

  • Design screen based prototypes.
  • Create screens that are easy to understand and use.

LIMITATIONS

These guidelines are merely suggestions which cover the basics of mobile interface design - participants need to custom design their screens based on their own learnings.
